MBTA Stop Finder

MBTA transit information from MBTA developer API
This app also makes use of Google's geolocation API to get the current location's coordinates.

The app demonstrates the use of a publicly available API . We request data using the XMLHttpRequest object from the MBTA api regarding the subway system. We get a JSON response that is used to display the routes in a table

Double click on any row with a Stop name to see it on a Google Map
